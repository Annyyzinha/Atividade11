INÍCIO SOBRE O PROJETO

Desvendando o Front-end com HTML e CSS na UTFPR!

Neste projeto da disciplina de Front-end da UTFPR, estou explorando os fundamentos essenciais para a construção de interfaces web. Aqui, aplico conceitos de **Estrutura (HTML)** e **Estilização (CSS)**.

Com o **HTML**, aprendi a organizar o conteúdo de uma página, utilizando tags semânticas para dar significado aos elementos, como cabeçalhos, parágrafos, links e seções.

Já com o **CSS**, consegui dar vida e estilo a esses elementos, controlando cores, fontes, layouts com Flexbox e Grid, e criando efeitos visuais modernos. Este trabalho é um passo importante na minha jornada como desenvolvedora Front-end.
